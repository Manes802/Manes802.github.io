Je vous explique les pointeurs en C

Les pointeurs sont des variables un peu différentes des autres variables. Un pointeur est une variable qui a pour valeur l’adresse d’une autre variable. On dit généralement qu’un pointeur pointe sur une variable. Pour faire une analogie, on dira que le pointeur est comme un téléphone, ou plus précisément une carte SIM, dans lequel on insert l’adresse de nos variables et non pas nos variables elles-même. Tout comme la carte SIM stock l'adresse de nos correspondants pour qu'on puisse les appeler, de même, le pointeur stock l’adresse d’une variable pour pouvoir ensuite l’appeler.

Exemple

Je crée une variable nommée taille_user de type double qui va contenir la taille de l’utilisateur de mon programme par exemple, et je crée aussi un pointeur nommé pointe_taille_user de type double (naturellement) qui pointe sur la variable taille_user.

Le code

double taille_user = 1.72; //déclaration et initialisation de la variable taille_user.
double *pointe_taille_user = &taille_user; //déclaration et initialisation du pointeur pointe_taille_user.

Comme montre le code, le pointeur contient l’adresse de la variable. Le symbole * permet juste de signifier que c’est un pointeur.

En mémoire nous aurons (sans tenir compte du nombre d’octet du type double) :

La variable (taille_user) Le pointeur (pointe_taille_user)
Adresse Valeur
……… Autre valeur
11A2E 1.72
11A2F Autre valeur
……… Autre valeur
Adresse Valeur
……… Autre valeur
15F91 11A2E
15F92 Autre valeur
……… Autre valeur

Hé oui ! Un pointeur a aussi le droit d’avoir une adresse : ce n’est pas parce qu’il contient une adresse qu’il ne doit pas avoir sa propre adresse.
Etant donné que le pointeur contient l’adresse d’une variable, il peut nous permettre de connaitre et de modifier la valeur de la variable dont il pointe (très avantageux).

Voici quelques syntaxes à connaitre sur les pointeurs
//déclaration d’un pointeur.
type (de la variable à pointer) *nom_du_pointeur ;
//CONSIDERONS LA VARIABLE DE L’EXEMPLE PRECEDENT.
double *pointe_taille_user ; /*déclaration d’un pointeur de type double. On peut également l’initialiser à NULL (ce qui est d’ailleurs conseillé) comme ceci double : *pointe_taille_user = NULL ;*/
pointe_taille_user ; /*n’est rien d’autre que la valeur du pointeur (adresse de la variable) donc pas de différence avec une variable simple. On peut l’utiliser dans un printf pour voir l’adresse de la variable.*/
&pointe_taille_user ; /*La encore comme vous pouvez le deviner, c’est l’adresse de notre pointeur, pareil avec les simples variables*/
*pointe_taille_user ; /*Seule différence. *pointe_taille_user (non précédé par le type) est lecontenu de la variable dont le pointeur pointe. En d’autre terme, *pointe_taille_user équivaut à taille_user.*/
